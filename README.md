# Kotlin starter project

* Gradle 6
* Kotlin 1.3.70
* Kotlin test & Junit 5

# Running Tests
Run tests with:
`gradle test`

# Build
Build with:
`gradle build`

Clean with:
`gradle clean`

# Opening the project
1. Open IntelliJ
1. Press 'get project from version control'
1. Enter the project url:
    git@gitlab.com:codesquad-nl/kata-base.git (with ssh key already set up)
    https://gitlab.com/codesquad-nl/kata-base.git
